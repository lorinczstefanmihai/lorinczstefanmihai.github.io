Education and my path pursuing this career


As I couldn't decide whether to pursue physics or informatics, I had chosen both. After high school, I wanted to learn something with a more interdisciplinary approach, so I applied for the Mechatronics specialization. It offers a broader understanding of engineering, making you a versatile professional. I always wanted to comprehend every system, starting from the hardware – understanding what every button does and how the case is made – to the instruction set inside the box. Furthermore, this field provides a dynamic and ever-changing environment, which keeps your skills relevant and up-to-date.

Line Follower Robot with PID and UI Control

This university-developed project features a fast and stable line follower robot using a PID control algorithm for precise tracking. An interactive UI enables real-time tuning of P, I, and D values for quick optimization. Designed for robotics contests, the system delivers smooth performance, rapid adaptation to different tracks, and reliable high-speed navigation.
